# EmploisQuebecSearchEngine

Outils de recherche pour 'Emplois Québec' et leur système de placement en ligne.

Comment utiliser : 

Ouvrir 'ng serve' in EmploisQuebecSearchEngine\src\webapp\EQSE

Ouvrir 'Request.java'

Ouvrir localhost:4200
